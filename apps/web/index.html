How It Works

Three things.
Done well.

01

AI That Actually Helps

Type naturally. Remind's AI parses your intent, suggests the right time, assigns priority, and refines the text. It works in the CLI and through Claude Code — your choice.

"tomorrow 5pm" "next friday" "in 2 hours" "end of month"
input   → "deploy by friday"
parsed  → Deploy by Friday
due     → Fri, 5:00 PM
priority → High
saved   → #42
02

Notifications That Won't Let You Forget

Native OS notifications with sound. Miss one? Remind nudges you again at 30 min, 1 hour, and 2 hours. Runs as a system service — survives reboots, uses zero resources when idle.

Zero Friction, Zero Bloat

No Electron. No browser. No Java runtime. Remind is a Python CLI that starts in milliseconds. Add a reminder and you're back to coding in two seconds flat. Search, filter by project, mark done — all without leaving your terminal.
