Features

Everything you need,
nothing you don't.

Remind is built for people who live in the terminal. No Electron apps, no browser tabs, no distractions.

01

AI-Enhanced Reminders

Type naturally. Remind's AI parses your intent, suggests optimal timing, assigns priority, and refines the text — all in milliseconds.

$ remind add "deploy by friday"
input   → "deploy by friday"
parsed  → Deploy by Friday
due     → Fri, 5:00 PM
priority → High
saved   → ✓ #42
02

Speak Naturally

Type dates like you think them. Remind uses dateparser to understand dozens of natural language date formats — no rigid syntax required.

"tomorrow 5pm" "next friday" "in 2 hours" "end of month" "monday 9am" "3 days from now"
~ remind add "standup" --due "tomorrow 9am"
✓ Saved #6 · Due: Tomorrow, 9:00 AM
~ remind add "ship hotfix" --due "in 2 hours" --priority high
✓ Saved #7 · Due: Today, 4:30 PM · High
~ remind add "quarterly review" --due "end of month"
✓ Saved #8 · Due: Feb 28, 5:00 PM
03

Notifications That Escalate

Native OS notifications with sound — AppleScript on macOS, D-Bus on Linux. Miss one? Remind nudges you again at 30 min, 1 hour, and 2 hours. Configurable intervals.

04

Platform-Native Scheduler

Runs as a proper system service — LaunchAgent on macOS, systemd on Linux. Survives reboots, uses zero resources when idle.

~ remind scheduler --enable
✓ Installed LaunchAgent
  ~/Library/LaunchAgents/
  com.remind.scheduler.plist
✓ Scheduler is running
05

Project Context

Group reminders by project with --project. Keep work tasks separate from personal. Filter, search, and report per-project.

~ remind add "fix auth bug" \
  --project acme-api \
  --priority high
✓ Saved #12 · acme-api
~ remind add "buy groceries" \
  --project personal
✓ Saved #13 · personal
06

Self-Diagnosing

remind doctor runs 4 health checks — database, config, license, and scheduler. Catches problems before you notice them.

~ remind doctor
✓ Database ····· OK
✓ Configuration ·· OK
✓ License ······· Pro
✓ Scheduler ····· Running
All systems healthy.
07

Usage Reports

remind report shows your stats — total reminders, completed this month, overdue count, AI quota used, and plan tier. JSON output for scripting.

How It Works

Three commands.
That's the whole API.

1

Install

One command. Homebrew, pip, or download the binary directly. Works on every major platform.

$ brew install remind-cli
2

Add

Write reminders in plain English. The AI handles parsing, timing, and priority for you.

$ remind add "task" --due "5pm"
3

Done

List, search, complete. Remind stays out of your way until you need it — then it's instant.

$ remind done 1
